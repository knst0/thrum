# knst0/thrum

> **⚠️ Warning: Early-stage / Experimental**
>
> This project is in active development and is **not production-ready**. APIs are unstable and may change significantly between versions. Use at your own risk.

A framework-agnostic node-and-edge graph library for building visual flow editors, pipeline builders, and diagram tools. Ships a headless core with a first-class SolidJS adapter.

## Installation

```shell
# npm, yarn, pnpm, bun
install @thrum/core @thrum/solid
```

## Stability

Expect breaking changes. This is a personal/experimental project and API surfaces will evolve rapidly. Not recommended for production use.
